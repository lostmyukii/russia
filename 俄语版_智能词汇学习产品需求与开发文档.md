# 俄语版百词斩智能词汇学习产品需求与开发文档

> 文档版本：1.0
> 生成日期：2026-06-14
> 产品方向：面向初中俄语、高中俄语学习者的百词斩式智能背词 Web/PWA
> 首期目标：完成“选册别 → 生成计划 → 学新词 → 主动回忆 → SRS 复习 → 错词强化 → 打卡 → 排行榜激励”的学习闭环
> 参考来源：`AGENTS.md`、`融合版_智能词汇学习产品需求分析.md`、`docs/DEVELOPMENT.md`

---

## 1. 文档定位

本文档用于指导“俄语版百词斩”的产品设计、开发拆解、词库建设和验收。它继承原智能词汇学习平台的核心原则：

- 用科学记忆和主动回忆提升长期记忆，而不是只做翻卡片；
- 先完成可运行、可测试、可持久化的纵向闭环；
- 服务端作为学习进度、排行榜积分和打卡结果的最终事实来源；
- 移动端优先，以 Web/PWA 作为首期形态；
- 不创建无法使用的空壳功能。

本版本针对俄语学习做了改进：

- 词库从英语考试扩展为“初中俄语 + 高中俄语”；
- 词汇按人教版教材、学段、年级、册别、单元、课次分层管理；
- 学习卡片支持西里尔字母、重音、转写、词性、性、数、格、动词体和变位提示；
- 首期加入“背单词排行榜”，但排名依据有效学习行为，避免用户靠无效刷题冲榜。

## 2. 产品愿景与定位

### 2.1 产品愿景

打造一款适合中国中学生使用的俄语智能背词工具，让学生在课本同步、考试复习和日常积累中，用更少时间记住更多俄语词汇，并能逐步做到“读得出、听得懂、拼得准、用得上”。

### 2.2 产品定位

| 维度     | 定位                                                                             |
| -------- | -------------------------------------------------------------------------------- |
| 产品类型 | 俄语词汇学习 Web/PWA                                                             |
| 核心用户 | 初中俄语学生、高中俄语学生、俄语教师、家长                                       |
| 核心场景 | 课本同步背词、单元复习、期中期末备考、高考俄语词汇巩固                           |
| 差异化   | 人教版初高中俄语按册按单元词库 + 俄语语法提示 + 主动回忆 + SRS 复习 + 排行榜激励 |
| 首期形态 | React + TypeScript 响应式 Web/PWA                                                |

### 2.3 核心学习闭环

```text
选择学段与册别
  → 生成每日计划
  → 学习俄语新词
  → 完成主动回忆题
  → 根据 SRS 安排复习
  → 错词进入强化队列
  → 完成打卡
  → 更新排行榜与学习看板
```

## 4. 俄语学习特性

俄语词汇学习不能直接照搬英语背词流程。首期需要支持以下俄语特征：

### 4.1 西里尔字母与拼写

- 单词展示使用俄文原形，如 `школа`、`говорить`；
- 支持重音标记字段，如 `шко́ла`、`говори́ть`；
- 拼写题使用俄语键盘输入，允许移动端调起系统键盘；
- 拼写判定默认忽略重音符号，但重音训练题单独校验重音位置。

### 4.2 发音与重音

- 每个词尽量提供标准发音音频；
- 无音频时可展示转写或读音提示；
- 重点词提供重音提醒；
- 听音辨词题在音频资源完整时启用。

### 4.3 词性、性、数、格

词卡片不只展示中文释义，还要展示俄语学习必需的语法信息：

- 名词：性、复数形式、常见格变化提示；
- 形容词：基本形式、阳性/阴性/中性提示；
- 动词：未完成体/完成体、常见体对、第一/第二变位提示；
- 代词、数词、前置词、连接词：展示常用搭配和例句。

### 4.4 主动回忆题型

MVP 题型应覆盖：

- 看俄语选中文释义；
- 看中文选俄语词；
- 听音选俄语词；
- 俄语拼写输入；
- 例句填空；
- 重音位置选择；
- 名词性别选择；
- 动词体选择。

## 5. 词库内容设计

### 5.1 词库组织原则

词汇内容必须按“人教版教材 → 学段 → 年级 → 册别 → 单元 → 课次 → 单词”组织。正式上线词表必须来自授权或可追踪来源，不在生产库中手工录入无来源内容。

单元分类以人教版俄语教材的实际单元为准。导入时必须提供教材版本、册别、单元编号、单元名称和课次；如果不同学校使用的版本或修订年份不同，应通过 `textbookVersion` 与 `source` 字段区分，而不是混入同一个词库。

```text
初中俄语
  ├── 七年级上册
  ├── 七年级下册
  ├── 八年级上册
  ├── 八年级下册
  ├── 九年级上册
  └── 九年级下册

高中俄语
  ├── 必修第一册
  ├── 必修第二册
  ├── 必修第三册
  ├── 选择性必修第一册
  ├── 选择性必修第二册
  ├── 选择性必修第三册
  ├── 选择性必修第四册
  └── 高考高频词与真题拓展
```

> 如果实际教材册别与上表不同，导入时以学校或授权教材版本为准。系统层面保留 `grade`、`volume`、`unit`、`lesson` 字段，可适配“上下册”“全一册”“必修/选择性必修”等不同版本。

### 5.3 单词内容字段

每个俄语词至少包含：

- `lemma`：俄语原形；
- `stressedLemma`：带重音形式；
- `partOfSpeech`：词性；
- `definitionZh`：中文核心释义；
- `bookSlug`：所属册别；
- `publisher`：教材出版社，MVP 固定为 `pep`；
- `textbookVersion`：人教版教材版本或修订年份；
- `unit`：单元；
- `unitTitle`：人教版教材单元名称；
- `lesson`：课次；
- `examples`：至少一个俄语例句和中文翻译；
- `audioUrl`：标准发音音频，可为空但需标记资源状态。

建议补充：

- `transcription`：读音提示或转写；
- `gender`：名词性；
- `pluralForm`：复数形式；
- `aspect`：动词体；
- `aspectPair`：体对；
- `conjugation`：动词变位类型；
- `caseHints`：常见格变化或搭配；
- `imageUrl`：图片或场景图；
- `source`：教材、授权词表或自建内容来源。

## 7. 背单词排行榜设计

### 7.1 设计目标

排行榜用于提升坚持率和班级氛围，但不能鼓励无效刷题。排名应围绕“真实掌握”和“稳定坚持”，而不是单纯统计点击次数。

### 7.2 排行榜类型

| 榜单   | 周期               | 排名对象       | 适用场景                 |
| ------ | ------------------ | -------------- | ------------------------ |
| 今日榜 | 每日               | 个人           | 当日完成任务后的即时反馈 |
| 周榜   | 周一至周日         | 个人/班级      | 形成持续学习节奏         |
| 赛季榜 | 7 天、21 天、30 天 | 个人/班级      | 运营活动和挑战赛         |
| 册别榜 | 按词库             | 同一册别学习者 | 比较同册学习进度         |
| 班级榜 | 按教师创建班级     | 班级学生       | B 端轻量预留             |

### 7.3 排名指标

建议使用“有效背词积分”，由服务端根据学习记录计算：

```text
有效背词积分 =
  新增掌握词数 × 10
  + 到期复习完成词数 × 3
  + 错词消灭词数 × 8
  + 连续打卡加成
  - 无效重复答题惩罚
```

积分规则：

- 同一单词当天重复刷正确，只计一次主要积分；
- 未到期复习的重复作答不进入核心积分；
- 答错后再次答对可以加少量修正分，但不能超过首次掌握分；
- 每日积分设置上限，防止异常刷榜；
- 排行榜展示“掌握词数、复习完成率、连续打卡”三个解释字段。

### 7.4 未成年人保护

- 默认只展示昵称、头像、积分和排名，不展示手机号、学校详细信息等敏感内容；
- 初中用户默认不开启陌生人全站榜，只开放同册匿名榜或班级榜；
- 班级榜由教师或班级邀请码加入；
- 支持用户隐藏自己在公开榜单中的展示；
- 异常行为需要从榜单中临时剔除并保留审计记录。

### 7.5 排行榜数据模型

```ts
type LeaderboardScope = 'daily' | 'weekly' | 'seasonal' | 'book' | 'class'

type LeaderboardSeason = {
  id: string
  scope: LeaderboardScope
  title: string
  bookId: string | null
  classId: string | null
  startsAt: string
  endsAt: string
  status: 'scheduled' | 'active' | 'closed'
}

type StudyScoreEvent = {
  id: string
  userId: string
  wordId: string | null
  sessionId: string
  eventType:
    | 'new_word_mastered'
    | 'review_completed'
    | 'mistake_removed'
    | 'checkin_streak_bonus'
    | 'invalid_repeat_penalty'
  scoreDelta: number
  occurredAt: string
  idempotencyKey: string
}

type LeaderboardEntry = {
  seasonId: string
  userId: string
  displayName: string
  avatarUrl: string | null
  score: number
  masteredWordCount: number
  reviewCompletionRate: number
  streakDays: number

```

## 8. 核心用户流程

### 8.1 首次学习流程

```text
打开产品
  → 登录或访客体验
  → 选择“初中俄语”或“高中俄语”
  → 选择年级与册别
  → 设置每日新词量和目标日期
  → 系统生成计划
  → 完成首组新词学习
  → 完成主动回忆题
  → 查看学习结果
  → 达到最低任务后打卡
  → 查看今日榜位置
```

### 8.2 日常学习流程

```text
进入首页
  → 查看今日到期复习和新词数量
  → 优先完成到期复习
  → 学习今日新词
  → 完成混合回忆题
  → 服务端更新掌握状态与复习时间
  → 计算有效背词积分
  → 完成打卡
  → 更新看板和排行榜
```

### 8.3 错词消灭流程

```text
答错或反应过慢
  → 记录错误类型
  → 单词进入错词本
  → 安排短间隔复习
  → 连续 3 次主动回忆正确
  → 标记错词消灭
  → 进入普通复习队列
  → 产生错词消灭积分
```

## 12. SRS v1 与俄语题型规则

### 12.1 回答等级

| 等级    | 判定                                         |
| ------- | -------------------------------------------- |
| `again` | 答错，或用户主动选择“不认识”                 |
| `hard`  | 答对但明显犹豫、使用提示，或响应超过题型阈值 |
| `good`  | 正常速度答对                                 |
| `easy`  | 快速且无提示答对                             |

### 12.2 题型阈值

| 题型            | `hard` 默认阈值 |
| --------------- | --------------- |
| 看俄选中        | 8 秒            |
| 看中选俄        | 8 秒            |
| 听音选词        | 10 秒           |
| 俄语拼写        | 18 秒           |
| 重音位置        | 12 秒           |
| 名词性别/动词体 | 12 秒           |
| 例句填空        | 20 秒           |

### 12.3 初始间隔

| 结果    | 下次复习  |
| ------- | --------- |
| `again` | 10 分钟后 |
| `hard`  | 1 天后    |
| `good`  | 2 天后    |
| `easy`  | 4 天后    |

### 12.4 错误类型影响

- 拼写错误：优先安排拼写题；
- 听音错误：优先安排听音辨词；
- 重音错误：增加重音位置题；
- 语法错误：增加性别、格、动词体或变位提示；
- 语境错误：增加例句填空和搭配题。

## 14. PWA 与离线边界

离线时允许：

- 打开已缓存应用；
- 查看最近一次今日任务；
- 继续已下载且未过期的学习会话；
- 将作答暂存到本地同步队列。

离线时不允许：

- 创建新的学习计划；
- 切换未缓存册别；
- 生成新的服务端学习会话；
- 将打卡显示为最终成功；
- 更新排行榜名次。

恢复网络后，客户端按创建时间提交作答队列。排行榜积分只能在服务端确认学习记录后更新。

当前 MVP 实现：

- Web 端提供“缓存离线学习包 -> 模拟断网作答 -> 恢复网络同步”的可验证流程；
- 离线学习包包含当前学习会话、词卡、主动回忆题、缓存时间和过期时间；
- 离线同步记录保存 idempotency key、创建时间、重试次数和最近错误；
- `/api/v1/offline/sync` 复用学习完成逻辑，同一 idempotency key 重复同步不会重复计分；
- PWA 运行时缓存仅覆盖教材词库接口，打卡和排行榜必须等待服务端确认。

- 初中俄语和高中俄语词库可按人教版教材册别与单元选择；

## 18. 风险与应对

| 风险             | 表现                   | 应对                                                         |
| ---------------- | ---------------------- | ------------------------------------------------------------ |
| 词库版权风险     | 直接复制教材词表或例句 | 使用授权来源，导入时记录 `source`                            |
| 俄语输入体验差   | 手机拼写题难输入       | 支持系统俄语键盘，允许选择题作为替代练习                     |
| 排行榜刷榜       | 用户重复刷简单题冲分   | 积分按有效掌握计算，同词同日去重，设置每日上限               |
| 未成年人社交风险 | 公开榜暴露学生信息     | 默认匿名展示，限制陌生人社交，支持隐藏排名                   |
| 学习效果弱       | 只记住卡片不记住单词   | 强制主动回忆，错词进入复习，题型轮换                         |
| 内容维护成本高   | 不同教材版本不一致     | 用出版社、教材版本、学段、年级、册别、单元、课次字段适配版本 |

## 20. 结论

俄语版百词斩的核心不是简单替换语言，而是围绕俄语本身的学习难点重新设计词库、题型和反馈机制。首期应聚焦初中俄语和高中俄语按册学习，通过主动回忆、SRS、错词强化、打卡和有效排行榜形成可坚持的学习闭环。

首版推荐交付重点：

```text
按册词库
  + 俄语专项词卡
  + 主动回忆
  + SRS 复习
  + 错词消灭
  + 打卡
  + 有效背词排行榜
```

这条闭环足够轻，也足够完整，可以支撑后续教师端、班级榜、AI 例句和高考俄语专项训练继续扩展。
